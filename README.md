# AutoGraph Service

Сервис для переноса данных из TXT/S2P файлов измерений в Excel таблицы.

## Описание

Сервис считывает данные из TXT или S2P файлов, парсит табличные данные и записывает их в указанный Excel файл.

Поддерживает режимы работы:
1. **Обычный режим** - создает новый лист с данными
2. **Режим шаблона TXT** - копирует шаблон листа расчета, вставляет данные и автоматически добавляет новую серию на лист с графиками
3. **Режим шаблона S2P** - копирует шаблон, вставляет S2P данные, копирует графики из шаблона и автоматически масштабирует оси графиков

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

### Базовое использование (обычный режим)

```bash
python -m autograph_service.main --txt <путь_к_TXT> --excel <путь_к_Excel>
```

### Режим шаблона TXT

```bash
python -m autograph_service.main --txt <путь_к_TXT> --excel <путь_к_Excel> --template
```

### Режим шаблона S2P

```bash
python -m autograph_service.main --s2p <путь_к_S2P> --excel <путь_к_Excel> --template
```

### Примеры

```bash
# Базовый пример (обычный режим)
python -m autograph_service.main --txt "NiPA-64-F-MD_1_18dBm_IMP_iPA-64-F_2026-03-03_17-42-42.txt" --excel "R086W2-2 различные образц Н.Р..xlsx"

# Режим шаблона TXT - копирует шаблон и добавляет данные
python -m autograph_service.main --txt "NiPA-64-F-MD_1_18dBm_IMP_iPA-64-F_2026-03-03_17-42-42.txt" --excel "R086W2-2 различные образц Н.Р..xlsx" --template

# Режим шаблона S2P - одиночный файл
python -m autograph_service.main --s2p "0283 30mA 1 obr.s2p" --excel "template.xlsx" --template

# Режим шаблона S2P - пакетная обработка папки
python -m autograph_service.main --s2p-dir ./s2p_files --excel "template.xlsx" --template

# С указанием имени шаблона (по умолчанию "№1")
python -m autograph_service.main --txt data.txt --excel template.xlsx --template --template-sheet "№1"

# С указанием имени листа для графиков (по умолчанию "Графики")
python -m autograph_service.main --txt data.txt --excel template.xlsx --template --charts-sheet "Графики"

# Без интерактивных вопросов (автоматический режим)
python -m autograph_service.main --s2p data.s2p --excel template.xlsx --template --no-prompt
```

### Аргументы командной строки

| Аргумент | Описание | Обязательный |
|----------|----------|--------------|
| `--txt` | Путь к TXT файлу с данными измерений | Да (или --s2p/--txt-dir/--s2p-dir) |
| `--s2p` | Путь к S2P файлу с S-параметрами | Да (или --txt/--txt-dir/--s2p-dir) |
| `--txt-dir` | Путь к папке с TXT файлами для пакетной обработки | Да (или --txt/--s2p/--s2p-dir) |
| `--s2p-dir` | Путь к папке с S2P файлами для пакетной обработки | Да (или --txt/--s2p/--txt-dir) |
| `--excel` | Путь к Excel файлу шаблона | Да |
| `--sheet` | Имя листа в Excel (по умолчанию: имя TXT файла) | Нет |
| `--output` | Путь для сохранения (по умолчанию: перезаписывает исходный) | Нет |
| `--template` | Включить режим шаблона (копирование шаблона + графики) | Нет |
| `--template-sheet` | Имя листа-шаблона для копирования (по умолчанию: "№1") | Нет |
| `--charts-sheet` | Имя листа с графиками для TXT (по умолчанию: "Графики") | Нет |
| `--charts-sheet-template` | Имя листа с графиками в шаблоне для S2P (по умолчанию: "Графики") | Нет |
| `--create-chart` | Создать новый график. Формат: `--create-chart <Y_колонка> <Буква_колонки> <Заголовок_Y_оси>` | Нет |
| `--no-prompt` | Отключить интерактивные вопросы (использовать значения по умолчанию) | Нет |

#### Создание новых графиков

Для добавления пользовательских графиков используйте аргумент `--create-chart`:

```bash
# Создать график Pвых Вт (колонка C)
python -m autograph_service.main --txt data.txt --excel template.xlsx --template \
    --create-chart Pout_W C "Pвых, Вт"

# Создать несколько графиков
python -m autograph_service.main --txt data.txt --excel template.xlsx --template \
    --create-chart Pout_W C "Pвых, Вт" \
    --create-chart Pout_dBm K "Pвых, дБм" \
    --create-chart Id_A S "Id, А"
```

Доступные типы данных для Y оси:
- `Pout_W` - Pвых, Вт (колонка U в листе данных)
- `Pout_dBm` - Pвых, дБм (колонка P)
- `Id_A` - Id, А (колонка R)
- `Ig_mA` - Ig, мА (колонка Q)
- `Gain_dB` - Gain, дБ (колонка S)
- `Efficiency` - КПД, % (колонка T)

## Формат входного TXT файла

Файл должен содержать:
1. Метаданные в начале файла (ключ-значение, разделенные табуляцией)
2. Строку заголовков таблицы (разделенные табуляцией)
3. Строки данных таблицы (разделенные табуляцией)

Пример:
```
Мощность, dBm	18,000000
Gate start, ms	20,000000
...
№	f, MHz	P, dBm	IG, mA	ID, A	Gain, dB	КПД, %	Pвых, W
1	500000000	33,513217	0,000049	0,140955	15,513217	5,529739	2,245545
...
```

## Режим шаблона TXT

В режиме шаблона TXT сервис:
1. Копирует указанный лист-шаблон (например, "№1") в новый лист с именем на основе TXT файла
2. Записывает данные из TXT в колонки A-H нового листа
3. Записывает вычисляемые значения в колонки P-U (PdBm, Ig, Id, Gain, КПД, Pвых)
4. Добавляет новую серию на лист "Графики" с формулами, ссылающимися на новый лист
5. Обновляет метаданные (мощность, ток покоя, напряжение и т.д.)

**Важно:** Для корректной работы режима шаблона TXT в Excel файле должны присутствовать:
- Лист-шаблон (по умолчанию "№1" или "№2") с таблицей и формулами
- Лист "Графики" с настроенными графиками и ссылками на данные

## Режим шаблона S2P

В режиме шаблона S2P сервис:
1. Копирует весь Excel файл шаблона в новый файл с именем на основе S2P файла
2. Копирует указанный лист-шаблон (например, "№1") в новый лист
3. Записывает данные из S2P в колонки A-E нового листа (Freq, S11M, S21M, S12M, S22M)
4. Копирует лист с графиками из шаблона
5. Обновляет диапазоны данных в графиках
6. Автоматически масштабирует оси графиков на основе анализа S2P данных

**Интерактивные вопросы:**
При запуске программа задает вопросы для уточнения параметров:
- Имя выходного файла (по умолчанию: имя S2P файла + .xlsx)
- Имя листа-шаблона (по умолчанию: "№1")
- Копировать лист с графиками? (Да/нет)
- Имя листа с графиками в шаблоне (по умолчанию: "Графики")
- Имя нового листа с графиками (по умолчанию: "S2P Графики")

**Автоматический режим (--no-prompt):**
Использует все значения по умолчанию без вопросов.

**Пакетная обработка (--s2p-dir):**
- Сканирует папку на наличие S2P файлов
- Группирует файлы по 8 штук на Excel файл
- Создает отдельные Excel файлы для каждой группы
- Графики создаются один раз на группу с объединенным масштабированием

**Важно:** Для корректной работы режима шаблона S2P в Excel файле должны присутствовать:
- Лист-шаблон (по умолчанию "№1") с таблицей для S2P данных
- Лист "Графики" с настроенными графиками для S-параметров (S11, S21, S12, S22)
